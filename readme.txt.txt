petunjuk penggunaan:


WINDOWS:
=============pingserver=================


 a. buka cmd sebagai admin
    > klik start, search CMD
    > klik kanan CMD, run as administrator
 b. buka direktori tempat penyimpanan pingserver.py
	misal: > saat ini anda berada pada C:\Windows\System32
	       > anda menyimpan pingserver.py di C:\Users\NamaSaya\Desktop
	       > maka ketik "cd C:\Users\NamaSaya\Desktop" agar anda masuk ke direktori tsb
 c. ketik "python pingserver.py"
 d. biarkan program berjalan	
!!!!BIARKAN PROGRAM BERJALAN AGAR STATUS SERVER TERUS TERUPDATE!!!!

LINUX(tested with ubuntu):
=============pingserver=================


 a. buka terminal dengan otoritas admin
    >tekan Alt+F2, lalu dialog "Run Application" akan muncul
    >ketik "gnome-terminal" pada dialog dan tekan "Enter". Maka akan terbuka terminal baru tanpa otoritas admin
    >Sekarang, di terminal yang baru, ketik "sudo gnome-terminal"// Anda akan diminta passwordnya, 
    >masukkan password lalu tekan "Enter". (password seakan2 tidak terinput, jangan pedulikan, ketikkan saja password lalu tekan enter)
 
 b. buka direktori tempat penyimpanan pingserver.py
	misal: > saat ini anda berada pada root@NamaUser:
	       > anda menyimpan pingserver.py di /Desktop/MagangDSI/multiIPPinger
	       > maka ketik "cd Desktop/MagangDSI/multiIPPinger" agar anda masuk ke direktori tsb
 c. ketik "python pingserver.py"
 d. biarkan program berjalan	
!!!!BIARKAN PROGRAM BERJALAN AGAR STATUS SERVER TERUS TERUPDATE!!!!
